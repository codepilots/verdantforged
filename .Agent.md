# VerdantForged

A peer-to-peer marketplace where three agents — a **Skill Provider**, a **Requester**, and an **Environment Broker** — collaborate to execute code with neither party exposing their IP or data. The Broker runs an attested NemoClaw SEV-SNP enclave. Payment releases only after verified execution.

## What the protocol does, in 30 seconds

```
1. Agreement (P2P, off-chain)
   Provider → Requester: SkillManifest + BrokerAttestationRequirements
   Requester → Provider: Accept + DataHash + ResultPubkey + MPPEscrowID

2. Attestation Verification (Requester → Broker)
   Requester verifies Broker's attestation matches expected measurements

3. Encrypted Payload (Requester → Broker)
   Requester sends: EncryptedSkill + EncryptedData + ResultPubkey + MPPEscrowID

4. Enclave Execution (inside NemoClaw)
   - Decrypt skill + data
   - Verify skill hash
   - Execute (fuel-limited, no syscalls)
   - Encrypt result to ResultPubkey
   - Call MPP release (only allowed egress)
   - Return encrypted result

5. Result + Payment
   Broker → Requester: EncryptedResult
   Requester decrypts
   MPP releases payment to Skill Provider
```

## Security guarantees (the table to memorize)

| Attack | Mitigation |
|--------|------------|
| Broker reads skill source | Skill encrypted to enclave key; attestation shows no debug interface |
| Broker reads requester data | Same — data encrypted to enclave |
| Broker copies skill for later | Enclave ephemeral; no persistent storage; measured boot |
| Broker sends result to attacker | Result encrypted to requester's pubkey; enclave has no other key |
| Broker runs different code | Attestation measurement = hash of approved skill-runner binary |
| Broker outputs to network | OpenShell policy: ONLY MPP egress allowed |
| Broker stalls indefinitely | Fuel limit (instruction count) + wall-clock timeout enforced by NemoClaw |

- Simulated MPP payment release (log only) — production needs real Stripe API call with scoped key
- Mock attestation provider — production needs real SEV-SNP `/dev/sev/guest`

- No constant-time crypto guarantee
